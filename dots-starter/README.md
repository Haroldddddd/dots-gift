# 点点 Dots Starter

> 点点 AI 助手 Demo 开发模板。用 Cursor 打开后直接开始 vibe coding，AI 会自动遵循设计规范。

## 快速开始

1. **下载或克隆** 这个文件夹到本地
2. **用 Cursor 打开** → File → Open Folder → 选中 `dots-starter`
3. **预览组件** → 浏览器打开 `assets/preview.html`，看看所有组件长什么样
4. **开始创作** → 在 Cursor AI 对话框里描述你想做的功能

就这么简单。AI 会自动读取 `SKILL.md` 和 `references/` 里的设计规范和组件库。

## 文件结构

```
dots-starter/
├── SKILL.md                          ← 技能入口（AI 自动读取）
├── references/
│   ├── design-system.md              ← 设计规范（颜色/字号/间距/圆角/动画/富文本）
│   └── components.md                 ← 组件库（14 个组件的 HTML + CSS 代码片段）
├── assets/
│   ├── css/
│   │   ├── tokens.css                ← 设计变量（颜色/间距/圆角/字体）
│   │   ├── reset.css                 ← 基础重置
│   │   ├── app-shell.css             ← 框架层样式（状态栏/导航/输入栏）
│   │   └── content.css               ← 内容区样式（你的代码写这里）
│   ├── img/                          ← 图片资源（气泡尾巴/图标/状态栏等）
│   ├── lottie/
│   │   └── cloud.json                ← 思考中云朵呼吸动画
│   ├── index.html                    ← 主页面（框架已搭好，从这里开始）
│   └── preview.html                  ← 组件预览页（浏览器打开查看效果）
└── README.md
```

## 三层规范，各管各的

| 文件 | 谁读 | 管什么 |
|------|------|--------|
| `SKILL.md` | AI 自动读取 | 技能入口，核心规则速查，禁止清单 |
| `references/design-system.md` | AI 自动读取 | 颜色 Token、字号体系、间距节奏、圆角层级、动画规范、富文本排版规则、组件 Token 映射 |
| `references/components.md` | AI 自动读取 | 14 个组件的 HTML + CSS 代码片段，拿来即用 |

你不需要读这些文件，AI 会自己读。

## 框架层 vs 内容层

- **框架层**（不需要动）：状态栏、导航栏 "点点"、底部输入栏、安全区
- **内容层**（你的创作区）：`#content-area` 里的所有内容

## 设计规范速查

| 要素 | 值 |
|------|-----|
| 设备画布 | 430 × 932px（iOS 大屏） |
| 页面背景 | `--bg` #F3F3F3 |
| AI 气泡 | `--fillA` 白色 + `--separator2` 边框 |
| 用户气泡 | `--fillB` #E6E6E6 + `--separator` 边框 |
| 卡片背景 | `--fill5` 白色 80% |
| 品牌绿 | `--info5` #6FD2BD |
| 文字标题 | `--title` #141414 |
| 气泡/卡片圆角 | 22px |
| 富文本卡片圆角 | 36px |
| 按钮圆角 | 24px |
| 间距节奏 | 32 → 22 → 16 → 8 |
| 字体 | PingFang SC |
| 行高倍数 | ×1.7 |
| 字间距 | +6% |

## 注意事项

- 填充按钮只用 **品牌绿**（#6FD2BD），**禁止黑色和红色**
- 红色仅用于底部弹窗的危险操作文字
- 复选框/单选框统一为 **圆形**
- 内容卡片统一为毛玻璃「左文右图」结构，标题单行截断
- 思考中动画使用 Lottie（需引入 lottie-web），不用 CSS 动画
- 暗色模式变量已在 `assets/css/tokens.css` 中定义

## 怎么和 AI 对话

打开 Cursor 的 AI 对话框，直接描述你想做的功能即可。比如：

- "帮我做一个对话页面，用户问'推荐上海美食'，AI 回复一段富文本 + 3 张内容卡片"
- "加一个底部弹窗，有分享、收藏、删除三个选项"
- "做一个思考中状态，3 秒后自动变成 AI 回复"

AI 会自动参考设计规范和组件库来生成代码。
